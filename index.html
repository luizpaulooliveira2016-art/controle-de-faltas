❓ Como usar

📋

Guia Rápido

Aprenda a usar cada função

➕

Botão + (Flutuante)

Toque no botão azul "+" no canto inferior direito para cadastrar produtos, registrar faltas, vencimentos ou trocas.

📦

Cadastrar Produto

Busque online pelo nome ou código de barras. Preencha nome, marca, peso, tipo, fornecedor e imagem.

🚫

Registrar Falta

Selecione um produto cadastrado e informe estoque atual e sugestão de pedido. Use o scanner para encontrar rápido.

📅

Registrar Vencimento

Selecione o produto, informe quantidade, data de validade e localização na loja.

🔄

Registrar Troca

Selecione o produto, informe quantidade, motivo da troca e localização.

📷

Scanner

Use a câmera para ler códigos de barras. Disponível no cadastro, faltas, vencimentos e trocas.

📦

Marcar como Pedido

Na lista de faltas, marque produtos que já foram pedidos. O botão fica vermelho após confirmação.

✅

Troca Realizada

Na lista de trocas, confirme quando a troca foi realizada. O card fica vermelho.

📄

Relatórios

Gere PDFs ou compartilhe via WhatsApp. Use filtros por marca, fornecedor ou data.

🔔

Notificações

Receba alertas em tempo real quando outro aparelho fizer alguma ação no sistema.

🔄

Sincronização

Os dados são sincronizados automaticamente entre todos os aparelhos conectados.
